Availability
------------

This is a binary distribution of Versile Java, an implementation of
Versile Platform for Java SE 6 and later. Source distribution can be
downloaded from https://github.com/versiledev/versile-java and the latest
binary version of this software can be downloaded from
http://www.sci4all.org/versile/ .

For any re-distribution of this binary distribution or any derived
versions, ensure the conditions of this program's license for such
distribution are fully understood, including the requirements to
provide access to associated source code when licensed as LGPL.


Installation and Usage
----------------------

This binary distribution comes in the form of a JAR file which can be
directly imported into Java programs.

Before linking to the JAR file, ensure the implications of the license
conditions of this program are fully understood.
